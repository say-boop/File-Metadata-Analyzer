# 📄 File Metadata Analyzer

Утилита для извлечения скрытых метаданных из файлов различных форматов. Поддерживает изображения, PDF, DOCX и аудиофайлы.

---

## ✨ Возможности

- 📷 **Изображения (JPG/PNG)** — GPS-координаты, модель камеры, дата съёмки, ISO, выдержка, вспышка, ориентация, баланс белого, режим съёмки, серийный номер, объектив, ПО обработки
- 📕 **PDF** — автор, программа-создатель, страницы, заголовок, тема, вложения, извлечённый текст, поля форм
- 📝 **DOCX** — автор, даты, ревизии, параграфы, комментарии, стили, изображения, таблицы, секции
- 🎵 **Аудио (MP3/FLAC/WAV)** — исполнитель, альбом, жанр, длительность, битрейт, обложка, текст песни, количество каналов
- 🌐 **Веб-интерфейс** — сканирование папок через браузер
- 📄 **Экспорт отчётов** — JSON и HTML с картой GPS-координат
- 🗺️ **Карта** — отображение мест съёмки на Leaflet.js
- 💻 **EXE/DLL** — версия, компания, дата компиляции, разрядность (32/64 bit)
- 📦 **ZIP архивы** — список файлов, метод сжатия, размер до/после
- 🎬 **Видео (MP4/MKV)** — кодеки, разрешение, битрейт, субтитры
- 🔍 **Детальный просмотр** — клик по файлу раскрывает панель со всеми метаданными
- 🗺️ **Карта в интерфейсе** — GPS-метки отображаются на карте прямо в панели деталей
- 🖼️ **Предпросмотр** — миниатюры изображений, проигрывание аудио и видео

## 🛠️ Технологии

| Технология | Назначение |
|------------|------------|
| 🐍 **Python 3.12** | Язык разработки |
| ⚡ **FastAPI** | Веб-интерфейс |
| 🎨 **Jinja2** | HTML-шаблоны |
| 🗺️ **Leaflet.js** | Карта в HTML-отчёте |
| 📷 **exifread** | Извлечение EXIF из изображений |
| 📕 **PyPDF2** | Метаданные PDF |
| 📝 **python-docx** | Метаданные DOCX |
| 🎵 **mutagen** | Метаданные аудиофайлов |
| 🖼️ **Pillow** | Работа с изображениями |
| 🧪 **pytest** | Тестирование (26 тестов) |
| 📦 **Poetry** | Управление зависимостями |
| 💻 **pefile** | Анализ EXE/DLL |
| 🎬 **pymediainfo** | Метаданные видеофайлов |

## 🚀 Установка и запуск

```bash
git clone https://github.com/say-boop/File-Metadata-Analyzer.git
cd File-Metadata-Analyzer
poetry install
```

## 💻 Запуск веб-интерфейса
```bash
poetry run uvicorn app.main:app --reload
```
Открыть в браузере: http://localhost:8000

---

## 📡 API Endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| **GET** | / | Веб-интерфейс |
| **POST** | /scan | Сканировать папку (JSON: {"directory": "путь"}) |
| **GET** | /report/json | Скачать JSON-отчёт |
| **GET** | /report/html | Скачать HTML-отчёт с картой |
